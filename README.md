# StepCounter-done-

### Описание:
В данном приложение юзеру дают возможность отслеживать статистику по шагам, калориям, дистанции и затраченному времени с помощью апки здоровье от епл. Юзеру доступны 10 ачивок по количеству шагов за день, результатом может делиться в виде скрина апки. В настройках юзер может изменять свои цели по всем параметрам и редактировать свой профиль.
